Frozen Delights

The Frozen Delights Ordering System is a Java-based application designed to simulate an interactive ice cream ordering experience. This console and GUI-based program allows users to order from a variety of ice cream flavors, view their current orders, update or cancel orders, and proceed to payment, culminating in the display of a detailed receipt.

Inmate Information Management System

My team and I developed a GUI called the Inmate Information Management System using Java Swing, which connects to a database managed with XAMPP. This project involved collaboratively designing an intuitive interface where users can input, update, and retrieve inmate records efficiently. By integrating Java Swing with a MySQL database, we learned how to handle database connections, perform SQL queries, and ensure data consistency within the application. This group effort significantly improved our skills in both GUI design and database management.

SoleSavvy

I designed and developed a website called SoleSavvy using HTML, CSS, and Bootstrap entirely on my own. SoleSavvy serves as an online platform dedicated to shoes, where users can browse and select from a variety of high-quality footwear options. I focused on creating a visually appealing and user-friendly interface that highlights different shoe collections effectively. Moreover, I integrated a seamless purchasing process, allowing customers to complete their transactions directly on the website. This project not only enhanced my proficiency in web development but also underscored the significance of user experience and functionality in online retail environments.
